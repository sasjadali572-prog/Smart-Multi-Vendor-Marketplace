# Smart Multi-Vendor Marketplace

ADBMS Semester Project developed using MongoDB, Express.js, React.js, and Node.js.

## Features

- Product Management (CRUD)
- Vendor Analytics
- Revenue Analytics
- Order Management
- MongoDB Aggregation Pipelines
- MongoDB Views
- Indexing and Query Optimization
- MongoDB Atlas Integration
- Transactions and Stock Validation

## Technologies Used

- MongoDB Atlas
- MongoDB Compass
- MongoDB Shell
- Express.js
- React.js
- Node.js
- Recharts
